# Accessibility Audit

Every grab carries a11y context for the target element, and the FAB ships a page-wide scanner you can run on demand.

## Per-grab: `A11yInfo`

When an element is grabbed, its result includes:

```ts
interface A11yInfo {
  attributes: A11yAttribute[]; // ARIA + role + structural attrs
  role?: string;
  accessibleName?: string; // computed AccName
  accessibleDescription?: string;
  severity?: A11ySeverity; // "ok" | "warn" | "error"
  audits?: A11yAuditItem[]; // rule matches for this element
}
```

- `hasA11yAttributes()` is the fast path used on mousemove — it only asks "does this element have _any_ a11y-relevant attribute?"
- `extractA11yInfo()` runs the full extraction on grab

## Page-wide scan: `scanPageA11y()`

The FAB's **A11y panel** calls `scanPageA11y()` to walk every Vue component in the tree and run five built-in rules:

| Rule                    | Flags                                                               |
| ----------------------- | ------------------------------------------------------------------- |
| **Missing label**       | Form controls without `<label>`, `aria-label`, or `aria-labelledby` |
| **Missing alt text**    | `<img>` without `alt` (and not `role="presentation"`)               |
| **Missing button name** | `<button>` with no text, `aria-label`, or `aria-labelledby`         |
| **Empty link**          | `<a>` with no accessible name                                       |
| **Invalid ARIA role**   | `role="…"` set to a value not in the ARIA role list                 |

The panel groups violations by Vue component so you can jump to the component that owns the offending elements.

## Running the scan

Open the FAB and switch to the **A11y** panel — clicking "Scan" walks the Vue component tree and renders a grouped report (one block per component with findings).

## Tips for agent hand-off

Paste `result.a11y` into your chat prompt — agents read ARIA attributes directly and will propose fixes. For whole-page audits, pipe `scanPageA11y()` to JSON and paste the grouped summary.
